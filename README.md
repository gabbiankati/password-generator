# Password Generator

Um gerador de senhas desenvolvido em Java que cria senhas aleatórias utilizando a classe `SecureRandom`.

## Funcionalidades

* Solicita ao usuário o tamanho da senha.
* Valida a entrada para aceitar apenas números inteiros.
* Permite senhas de **1 a 60 caracteres**.
* Gera senhas utilizando letras maiúsculas, minúsculas, números e caracteres especiais.
* Utiliza `SecureRandom` para gerar valores aleatórios mais seguros.

## Tecnologias

* Java
* `SecureRandom`
* `Scanner`
* `StringBuilder`

## Como executar

1. Clone o repositório.
2. Compile o projeto.
3. Execute a classe `PasswordGenerator`.
4. Informe o número de caracteres desejado.
5. A senha será gerada e exibida no terminal.

## Exemplo

```text
Digite quantos caracteres você gostaria de ter em sua senha:
12

Senha gerada:
aB7@Km9#Q2$x
```
